**WC Buy Now Button** is a very useful plugin for adding a Buy Now Button in the WooCommerce Store with more functionality.

== About WC Buy Now Button ==

**WC Buy Now Button** is a very useful plugin to add Buy Now Button in WooCommerce Store . WC Buy Now Button plugin has more functions with all things of add Buy Now Button text change, Color Change, Background Change, Hover Background and color change. You can choose Buy Now Button add or not add in Single product page and Shop page.
Everybody can use this plugin on their WooCommerce websites. If you want to add Buy Now button in your WooCommerce Store for checkout the product easily, you can use this plugin.
"**WC Buy Now Button**" is easy to use.
**Note:)** You need to install WooCommerce plugin before installing "WC Buy Now Button" plugin.

== How to use ==
You can use this plugin with some steps, like:

* Search and Install the WC Buy Now Button Plugin. You will be redirected to the plugin admin page after installing it.
* You can change the Buy Now Button text change option, Customize the Buy Now Button and more options.
*  Then you go to your WooCommerce Website and you will see the Buy Now Button in single product page and shop page besed on your choose.

NOTE:) Must be you need to install WooCommerce Plugin before install the WC Buy Now Button plugin.
